# GoodNotes 영어 단어 스터디 세트 생성기

단어책 텍스트 또는 사진을 업로드하면 AI(Gemini)가 자동으로 단어, 뜻, 예문, 유의어, 반의어를 추출해 굿노트 스터디 세트 CSV를 생성해드립니다.

## 기능

- ✏️ **텍스트 입력**: 단어장 텍스트를 붙여넣으면 AI가 자동 파싱
- 📷 **이미지 업로드**: 단어책 사진/스캔 이미지 → AI Vision으로 자동 추출
- 🔄 **배치 처리**: 여러 장의 이미지를 자동으로 나눠서 처리
- 📥 **굿노트 호환 CSV**: 바로 임포트 가능한 형식으로 다운로드

## 사용 방법

1. 텍스트를 붙여넣거나 단어책 이미지를 업로드
2. 카드 구성 옵션 선택 (앞면/뒷면 내용, 언어 등)
3. **AI로 스터디 세트 생성** 클릭
4. CSV 다운로드 후 굿노트 앱에서 가져오기

## 굿노트 임포트 방법

1. 다운로드한 `.csv` 파일을 iPhone/iPad의 파일 앱 또는 iCloud Drive로 옮기기
2. 굿노트 앱 → 라이브러리 → **+ 새로 만들기** → **가져오기**
3. CSV 파일 선택 → 스터디 세트 자동 생성

> ⚠️ 굿노트 CSV 임포트는 iOS/iPadOS에서만 지원됩니다.

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- Google Gemini 2.0 Flash API (Vision + Text)
- No backend, no server — 100% 브라우저에서 실행
